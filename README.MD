## Install AWS

```
$ curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
```

### Verificar instalação

```
aws --version

```

## Configuração

```
aws configure

```

**gerar chave de acesso ao usuário e configurar access key id, secret access key e região**
